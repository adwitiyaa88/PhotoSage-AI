# PhotoSage AI

PhotoSage AI is a desktop application that helps users organize, clean, and manage large photo libraries using AI.

## Current Features

- Apple Photos integration
- Thumbnail gallery
- Interactive photo viewer
- Responsive image viewer

## Planned Features

- Google Photos integration
- Duplicate detection
- Similar photo finder
- Blur detection
- Screenshot cleaner
- AI recommendations

## Tech Stack

- Python
- PySide6
- Pillow
- Google Photos API
